> 一般用hydra爆破密码,用法[参考](/_posts/2019-03-13-Hydra)

| 端口 | 服务    | 漏洞                        |
| ---- | ------- | --------------------------- |
| 21   | FTP     | 匿名访问,弱口令             |
| 22   | SSH     | 弱口令                      |
| 23   | Telnet  | 弱口令                      |
| 80   | Web     | 常见Web漏洞或后台登录弱口令 |
| 161  | SNMP    | public弱口令                |
| 389  | LDAP    | 匿名访问                    |
| 443  | openssl | 心脏滴血等                  |
| 445  | smb     | 操作系统溢出漏洞            |
| 873  | rsync   | 匿名访问，弱口令            |
